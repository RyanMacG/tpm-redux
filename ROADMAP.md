# TPM Redux Roadmap

This document outlines the planned features and improvements for TPM Redux. Features are organized by priority and tracked in [GitHub Projects](https://github.com/RyanMacG/tpm-redux/projects).

## High Priority Features

### 1. Parallel Plugin Operations
**Status:** Planned
**Issue:** #TBD
**Description:** Run plugin install/update operations in parallel for faster execution

### 2. Lock File Support
**Status:** Planned
**Issue:** #TBD
**Description:** Generate and use lock files for reproducible plugin installations

## Medium Priority Features

### 3. Plugin Search and Discovery (Hybrid Approach)
**Status:** Planned - Approach Decided
**Issue:** #TBD
**Description:** Allow users to search for and discover tmux plugins using a hybrid approach (curated list + GitHub API)
